DSA Case Studies using AVL Tree, Segment Tree & BFS (Java)

This repository contains implementation and analysis of three Data Structures & Algorithms (DSA) case studies developed using Java.

The project demonstrates how advanced data structures are applied in real-world large-scale systems such as healthcare platforms, ride-sharing services, and social media networks.

Case Studies Included
1. MediFlow Hospital — BST & AVL Tree

Implementation of a patient indexing system using Binary Search Tree (BST) and AVL Tree.

Features:

AVL insertion and deletion
LL, RR, LR, RL rotations
Height balancing
Patient ID indexing
Performance comparison between BST and AVL

Complexity:

Search: O(log n)
Insert: O(log n)
Delete: O(log n)
2. Uber Bengaluru — Segment Tree with Lazy Propagation

Implementation of a surge pricing backend using Segment Tree + Lazy Propagation.

Features:

Range updates
Range maximum queries
Lazy propagation optimization
Real-time surge multiplier handling
Operation trace analysis

Complexity:

Range Update: O(log n)
Query: O(log n)
3. Twitter Retweet Reach — BFS Bounded Traversal

Implementation of Breadth First Search (BFS) for follower network analysis.

Features:

Queue-based BFS traversal
Depth-bounded search
Visited-set duplicate prevention
Graph reachability computation
Queue evolution analysis

Complexity:

Traversal: O(V + E)
